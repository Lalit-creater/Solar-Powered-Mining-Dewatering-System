
# Solar-Powered Mining Dewatering System

## Overview

This project presents a solar-powered automated dewatering system designed for closed-pit mining applications. The system monitors water levels and automatically controls a pump to remove accumulated water using renewable solar energy.

## Features

* Automated water-level monitoring
* Solar-powered operation
* Battery backup support
* Automatic pump ON/OFF control
* Real-time environmental monitoring
* Sustainable and eco-friendly design

## Components Used

* Arduino UNO
* Ultrasonic Sensor
* DHT11 Sensor
* Float Switch
* Relay Module
* Solar Panel
* Battery
* Water Pump
* LCD Display

## Working Principle

The ultrasonic sensor continuously measures the water level inside the mining pit. When the water level exceeds a predefined threshold, the Arduino activates the pump through the relay module. The float switch acts as a backup safety mechanism, while the solar panel and battery provide uninterrupted power to the system.
